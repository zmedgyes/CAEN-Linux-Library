Detailed Description

The RFIDDevice class is used to interact with RFID readers.

The RFIDDevice class offers a group of methods to work with the RFID readers. This class uses the RFIDMessage nad SerialModule classes to contact the reader and recieve response.

This class offers only a limited amount of already implemented commands for the RFID readers. Other commands can be implemented very quickly using the RFIDMessage construction methods.
